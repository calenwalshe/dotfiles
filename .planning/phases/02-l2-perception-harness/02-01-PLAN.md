**@suite decorated run() function:**

```python
@suite("l2_perception", tier="standard", default_schedule="0 * * * *", timeout_s=2400, tags={"perception", "catalog", "l2"})
async def run(params: dict) -> list[TestResult]:
```

Implementation flow:
1. Extract params: artifact_root, prune_days, capture_timeout_s, bypass_timeout_s
2. Call `_prune_artifacts(artifact_root, SUITE, prune_days)`
3. Generate run_tag: `time.strftime("%Y%m%dT%H%M%SZ", time.gmtime()) + "-" + uuid.uuid4().hex[:6]`
4. Generate run_id: `f"{SUITE}-{run_tag}"`
5. Connect to DB: `conn = await asyncpg.connect(DATABASE_URL)` -- use single connection, not pool (per research pitfall 4). Wrap entire run in try/finally to close connection.
6. Create Storage with a minimal pool (min_size=1, max_size=2) -- Storage requires a pool, not a bare connection. Create pool in try/finally.
7. Call `await storage.list_sites(active_only=True)` to get active catalog
8. For each site, wrap in try/except:
   a. Record start time
   b. Create browser session via aiohttp POST to `{BROWSER_SERVICE_URL}/sessions` with `{"name": f"l2-{uuid.uuid4().hex[:12]}", "url": site["base_url"]}`
   c. Navigate and capture screenshot (follow screenshot_tool.py HTTP call pattern)
   d. Check `nav.get("ok")` -- if False, record error TestResult and continue
   e. Extract page_title, page_text, screenshot_b64 from response
   f. Classify: `cls, conf = _classify_screenshot(page_title, page_text, site["base_url"])`
   g. If cls == "BLOCKED_CHALLENGE", attempt CF bypass under `_bypass_semaphore` with timeout
   h. Query previous classification BEFORE inserting current: `prev_scores = await storage.get_site_scores(site_key=site["site_key"], limit=1)`
   i. Detect drift: `drift_event = _detect_drift(prev_scores[0]["classification"] if prev_scores else None, cls)`
   j. Write artifacts: decode base64 PNG to `case_dir / "screenshot.png"`, write classification.json with url, site_key, timestamp, classification, confidence, challenge_detected, pre_bypass_class, drift
   k. Record to DB: `await storage.record_site_score(run_id=run_id, suite=SUITE, site_key=site["site_key"], passed=(cls=="PASS_CONTENT"), score=conf, classification=cls, challenge_detected=(cls=="BLOCKED_CHALLENGE"), nav_latency_ms=latency, content_confidence=conf, metadata={...})`
   l. Build TestResult and append to results list
   m. Delete browser session in finally block
9. Return results list

**Browser session lifecycle** (per screenshot_tool.py pattern):
- POST /sessions -> creates session
- POST /sessions/{name}/navigate -> navigates to URL, returns page content
- POST /sessions/{name}/screenshot -> captures screenshot_b64
- DELETE /sessions/{name} -> cleanup (in finally)
