```python
class AutonomyLevel(str, Enum):
    autonomous = "autonomous"
    supervised = "supervised"
    guided = "guided"

@dataclass
class RunConfig:
    autonomy_level: AutonomyLevel = AutonomyLevel.supervised
    token_budget: int = 100_000  # max tokens before circuit breaker
    time_budget_seconds: int = 600  # max wall-clock seconds
    max_gate_retries: int = 3  # max retries per gate before halt
    run_id: str = ""
```

Create CircuitBreaker:
```python
@dataclass
class CircuitBreaker:
    config: RunConfig
    tokens_used: int = 0
    start_time: float = field(default_factory=time.time)
    gate_eval_history: list[float] = field(default_factory=list)

    def record_tokens(self, count: int) -> None
    def check_budget(self) -> tuple[bool, str]  # (tripped, reason)
    def check_eval_plateau(self, current_score: float) -> tuple[bool, str]
    def should_halt(self) -> tuple[bool, str]
```

Eval plateau detection: if last 2+ eval scores are <= first score in the window (no improvement), plateau is detected.
