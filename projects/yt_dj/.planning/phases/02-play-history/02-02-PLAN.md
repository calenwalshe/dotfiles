  <files>agent-stack/liquidsoap/radio.liq</files>
  <action>
    In /home/agent/agent-stack/liquidsoap/radio.liq, insert the log_play function definition
    and the source.on_track call between the `map_metadata` block and the `output.icecast` call.

    The current radio.liq ends with:
    1. settings block
    2. playlist source
    3. fallback
    4. crossfade + fallback
    5. map_metadata block  ← insert AFTER this block
    6. output.icecast      ← keep as the LAST statement

    Insert these lines between map_metadata and output.icecast:

    ```liquidsoap
    # Play logger — fired on every track change via fire-and-forget shell-out
    # log_play.py exits 0 always; stream is never blocked or paused by this hook
    def log_play(meta) =
      filename = default("", list.assoc("filename", meta))
      title    = default("", list.assoc("title",    meta))
      artist   = default("", list.assoc("artist",   meta))
      thread.run(fun() ->
        ignore(system("/usr/bin/python3 /home/agent/projects/yt_dj/src/log_play.py " ^
                      process.quote(filename) ^ " " ^
                      process.quote(title)    ^ " " ^
                      process.quote(artist)))
      )
    end
    radio = source.on_track(synchronous=false, radio, log_play)
    ```

    CRITICAL:
    - `synchronous=false` is mandatory — the handler must never block the audio thread.
    - `thread.run` + `ignore(system(...))` is the fire-and-forget pattern. Exit code discarded.
    - `process.quote()` safely shell-escapes all three arguments (handles spaces, special chars in filenames).
    - The `filename` metadata field from Liquidsoap is the full absolute path — the same path used as `file_path` in `library_tracks` and `plays` tables.
    - The absolute path to python3 (`/usr/bin/python3`) is required inside the container (no PATH sourced).
    - The absolute path to `log_play.py` (`/home/agent/projects/yt_dj/src/log_play.py`) matches the :ro bind-mount path added in 02-01.
    - Do NOT change any other part of radio.liq.
  </action>
