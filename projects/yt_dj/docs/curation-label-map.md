# Curation Label Map — All 20 Genera

For any "grab more tracks for X" request, start here — not YouTube search.
Priority order: label Bandcamp catalog → Discogs wants validation → Bandcamp Daily.

**Priority tiers:**
- 🔴 CRITICAL — 0–1 holdings, essentially absent
- 🟠 HIGH — 2–4 holdings, undercovered relative to depth of scene
- 🟡 MEDIUM — 5–8 holdings, adequate starter coverage
- 🟢 SOLID — 9+ holdings or genre is intentionally shallow

## Usage

When a "grab more X" request comes in:

1. Find the genus in this map
2. Open the primary label Bandcamp page
3. Browse chronologically (oldest first for classics, newest for current underground)
4. Cross-check any candidates against Discogs: search release title → check "in 1 collection / X wants"
5. Pre-flight duration: `yt-dlp --print "%(duration)s %(title)s" <url>`
6. Propose 10–15 tracks as a named batch before downloading
