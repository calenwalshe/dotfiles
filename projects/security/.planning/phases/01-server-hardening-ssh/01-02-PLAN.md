# Plan 01-02: fail2ban + auditd

## Objective

Install and configure fail2ban (sshd jail covering ports 22 and 2022, banaction=ufw) and auditd. Enable and start both services.

## Tasks

- [ ] `sudo apt install -y fail2ban`
- [ ] Create `/etc/fail2ban/jail.local` with sshd jail covering ports 22,2022, bantime=3600, maxretry=5, banaction=ufw
- [ ] `sudo systemctl enable --now fail2ban`
- [ ] Verify: `sudo fail2ban-client status sshd`
- [ ] `sudo apt install -y auditd`
- [ ] `sudo systemctl enable --now auditd`
- [ ] Verify: `systemctl is-active auditd`

## Success Criteria

- `systemctl is-active fail2ban` returns `active`
- `fail2ban-client status sshd` shows jail enabled
- `systemctl is-active auditd` returns `active`
