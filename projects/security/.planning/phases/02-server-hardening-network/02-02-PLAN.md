# Plan 02-02: sysctl Hardening + UFW Logging

## Objective

Create `/etc/sysctl.d/99-hardening.conf` with network hardening params (rp_filter=1, send_redirects=0, log_martians=1, source_route=0, secure_redirects=0, tcp_rfc1337=1). Apply with `sysctl --system`. Verify Docker bridge interface retains rp_filter=0 (Docker manages it). Upgrade UFW logging from `low` to `medium`.

## Tasks

- [ ] Create `/etc/sysctl.d/99-hardening.conf` (do NOT set ip_forward=0 — Docker needs it)
- [ ] `sudo sysctl --system`
- [ ] Verify: `sysctl net.ipv4.conf.all.rp_filter` = 1, `send_redirects` = 0, `log_martians` = 1
- [ ] Verify Docker bridge: `sysctl net.ipv4.conf.docker0.rp_filter` still = 0
- [ ] `sudo ufw logging medium`
- [ ] Verify: `sudo ufw status verbose | grep Logging`

## Success Criteria

- `sysctl net.ipv4.conf.all.rp_filter` = 1
- `sysctl net.ipv4.conf.all.send_redirects` = 0
- `sysctl net.ipv4.conf.all.log_martians` = 1
- `ufw status verbose` shows Logging: medium
