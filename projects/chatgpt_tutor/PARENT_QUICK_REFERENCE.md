# Parent Quick Reference — Math Voice Tutor

Keep this handy during sessions. All commands are spoken aloud during voice mode.

## Commands

| Say this | What happens |
|----------|-------------|
| **"Parent mode on"** | Tutor explains directly to you (the adult) |
| **"Kid mode on"** | Tutor switches back to coaching the child |
| **"Hint only"** | One hint, not the answer |
| **"Next step only"** | Just the next step, nothing more |
| **"Check answer only"** | Correct or incorrect, minimal explanation |
| **"Make it easier"** | Tutor creates a simpler version first |
| **"Make it harder"** | Tutor creates a harder follow-up |
| **"Wrap up"** | Short celebration, what went well, one next skill |

## Session Checklist

- [ ] Crops ready? (individual problems cropped from worksheet)
- [ ] Child tried the first problem on her own for ~5 minutes?
- [ ] New chat started in the Math Tutor project?
- [ ] First crop uploaded?
- [ ] Session kickoff pasted?
- [ ] Voice started?

## During the Session

- **One crop per problem.** Upload a new crop when you move to a new problem.
- **You can type while voice is active.** If the tutor misreads something, paste the exact text — don't re-upload the image.
- **After each solved problem:** Ask your daughter to explain the solution back in her own words before moving on.
- **Verify the math yourself.** The tutor gets the approach wrong ~40% of the time. You are the accuracy backstop.

## Troubleshooting

| Problem | Fix |
|---------|-----|
| Tutor misreads a number or symbol | Paste the exact problem text into the chat (typing works during voice) |
| Tutor is too gushy/verbose | Say "Kid mode on" to reset, or tweak project instructions to be drier |
| Child is frustrated or stuck | Say "Make it easier" — tutor creates a simpler version |
| Tutor gives the answer too fast | Say "Hint only" or "Next step only" to slow it down |
| Tutor's math reasoning seems wrong | Say "Parent mode on", verify the approach yourself, correct if needed |
| Old context bleeding into new sessions | Delete old session chats from the project |
| Voice mode cuts off (Plus plan only) | Continue the session via text chat — type instead of speak |

## Key Numbers

- **~40%** of the time, ChatGPT's step-by-step math approach is wrong (arithmetic is usually fine, but the method can be off)
- **>95%** OCR accuracy on clean printed crops (drops below 60% for handwriting)
- **Pro plan:** unlimited voice — no time limit per session
- **Plus plan:** ~15 min/day Advanced Voice — keep sessions to 1-3 problems
