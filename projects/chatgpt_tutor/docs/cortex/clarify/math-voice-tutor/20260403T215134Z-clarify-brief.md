## Idea (verbatim)

The goal of this project is contained in the spec I shared: Math Voice Tutor Project Spec v1 — use one persistent ChatGPT Project for a 9-year-old daughter's grade-4 math tutoring via voice mode, with structured project instructions, session kickoff templates, image protocols, parent command modes, and privacy controls.

## Goal

A fully operational ChatGPT Project configured as a reusable voice-mode math tutor that lets a parent run guided, worksheet-based tutoring sessions for a 9-year-old with minimal per-session setup, accurate problem handling, and appropriate pedagogical scaffolding.

## Non-Goals

- Building a custom application or writing code — this is a ChatGPT Project configuration, not a software project
- Creating curriculum or selecting worksheets — worksheets are provided per-session by the parent
- Automating session creation or chat management — the parent manually creates each chat
- Training or fine-tuning any model — relies entirely on ChatGPT project instructions and prompting
- Supporting subjects other than math at this stage
- Building progress tracking, analytics, or session-over-session learning metrics
- Supporting multiple children with different profiles in the same project

## Constraints

- Must work within ChatGPT Projects feature set (project instructions, project-only memory, voice mode, image upload)
- Voice mode on iOS/Android only (subscriber feature)
- Project-only memory must be enabled at project creation — cannot be retrofitted
- Study Mode has known inconsistency (may give direct answers) — off by default
- Privacy: voice transcripts/images may train models unless "Improve the model for everyone" is disabled
- Child is 9 years old — all language, pacing, and interaction patterns must be age-appropriate
- Source-of-truth hierarchy is fixed: pasted text > crop > full page > read-aloud

## Assumptions

- The parent has a ChatGPT Plus/Pro subscription with voice mode access
- The parent is comfortable uploading worksheet photos and using voice mode on mobile
- Grade-4 math worksheets are the primary material (unit conversions, multiple choice, word problems, etc.)
- The parent will be present during sessions to use parent commands and manage flow
- One child, one project — no multi-child sharing needed
- The child can verbally interact with the tutor in English
- ChatGPT Projects feature and voice mode remain stable and available
