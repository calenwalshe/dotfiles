# Plan 02-01: Write and Register Claude Code Hooks

## Objective

Write three Claude Code hook scripts (SessionStart, PostToolUse, SessionEnd) that propagate W3C traceparent context and emit OTLP spans to Phoenix. Register them in `~/.claude/settings.json` without removing existing hooks. Install OpenLLMetry (`traceloop-sdk`).

## Execution Context

- Hook scripts go in: `~/.claude/hooks/`
- Settings file: `~/.claude/settings.json`
- OTLP endpoint: `http://localhost:6006/v1/traces`
- traceparent file: `/tmp/claude-trace-{session_id}.json`
- otel-cli PATH: `~/bin/otel-cli` — hooks run in non-interactive shell, must prepend PATH
- Existing hooks in settings.json must NOT be removed (additive writes only)
- Hook payload fields available via stdin JSON:
  - SessionStart: `session_id`, `transcript_path`
  - PostToolUse: `session_id`, `tool_name`, `tool_input`, `tool_response.output`, `tool_response.exit_code`, `cwd`, `context_window.remaining_percentage`
  - SessionEnd: `session_id`, `transcript_path`, `matcher`

### Task 2: Write SessionStart hook

Write `/home/agent/.claude/hooks/otel-session-start.sh`:

```bash
#!/usr/bin/env bash
# W3C traceparent init for Claude Code sessions
set -euo pipefail

export PATH="$HOME/bin:/usr/local/bin:/usr/bin:/bin"

# Read session_id from stdin
SESSION_ID=$(python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('session_id','unknown'))" 2>/dev/null || echo "unknown")

# Generate W3C traceparent: 00-{32hex}-{16hex}-01
TRACE_ID=$(openssl rand -hex 16)
SPAN_ID=$(openssl rand -hex 8)
TRACEPARENT="00-${TRACE_ID}-${SPAN_ID}-01"

# Write traceparent file
TRACE_FILE="/tmp/claude-trace-${SESSION_ID}.json"
python3 -c "
import json, sys
data = {
    'traceparent': '${TRACEPARENT}',
    'trace_id': '${TRACE_ID}',
    'session_id': '${SESSION_ID}',
    'created_at': '$(date -u +%Y-%m-%dT%H:%M:%SZ)'
}
print(json.dumps(data))
" > "${TRACE_FILE}"

# Emit root span to Phoenix
"$HOME/bin/otel-cli" span \
    --endpoint "http://localhost:6006/v1/traces" \
    --name "claude-session-start" \
    --traceparent "${TRACEPARENT}" \
    --attrs "session.id=${SESSION_ID},span.type=session-root" \
    2>/dev/null || true

exit 0
```

### Task 3: Write PostToolUse hook

Write `/home/agent/.claude/hooks/otel-post-tool-use.sh`:

The lazy-init guard MUST be present — handles the case where `/clear` was run (which ends the session and starts a new one without running SessionStart again in the same shell context).

```bash
#!/usr/bin/env bash
# Emit OTLP span per tool call with W3C traceparent propagation
set -euo pipefail

export PATH="$HOME/bin:/usr/local/bin:/usr/bin:/bin"

# Read payload fields from stdin
PAYLOAD=$(cat)
SESSION_ID=$(echo "$PAYLOAD" | python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('session_id','unknown'))" 2>/dev/null || echo "unknown")
TOOL_NAME=$(echo "$PAYLOAD" | python3 -c "import sys,json; d=json.load(sys.stdin); print(d.get('tool_name','unknown'))" 2>/dev/null || echo "unknown")

TRACE_FILE="/tmp/claude-trace-${SESSION_ID}.json"

# Lazy-init guard: generate traceparent if file absent (handles /clear edge case)
if [[ ! -f "$TRACE_FILE" ]]; then
    TRACE_ID=$(openssl rand -hex 16)
    SPAN_ID=$(openssl rand -hex 8)
    TRACEPARENT="00-${TRACE_ID}-${SPAN_ID}-01"
    python3 -c "
import json
data = {
    'traceparent': '${TRACEPARENT}',
    'trace_id': '${TRACE_ID}',
    'session_id': '${SESSION_ID}',
    'created_at': '$(date -u +%Y-%m-%dT%H:%M:%SZ)'
}
print(json.dumps(data))
" > "${TRACE_FILE}"
fi

# Read traceparent from file
TRACEPARENT=$(python3 -c "import json; d=json.load(open('${TRACE_FILE}')); print(d['traceparent'])" 2>/dev/null || echo "")

if [[ -z "$TRACEPARENT" ]]; then
    exit 0
fi

# Emit child span to Phoenix
"$HOME/bin/otel-cli" span \
    --endpoint "http://localhost:6006/v1/traces" \
    --name "claude-tool-${TOOL_NAME}" \
    --traceparent "${TRACEPARENT}" \
    --attrs "session.id=${SESSION_ID},tool.name=${TOOL_NAME}" \
    2>/dev/null || true

exit 0
```

### Task 6: Register hooks in ~/.claude/settings.json

Read the current settings.json, add the three hooks to their respective hook event arrays without removing existing hooks. The hooks key is structured as:

```json
{
  "hooks": {
    "SessionStart": [...existing..., {"type": "command", "command": "..."}],
    "PostToolUse": [...existing..., {"type": "command", "command": "...", "matcher": ""}],
    "SessionEnd": [...existing..., {"type": "command", "command": "..."}]
  }
}
```

Use python3 to read, merge, and write back atomically:

```python
import json

path = '/home/agent/.claude/settings.json'
with open(path) as f:
    settings = json.load(f)

if 'hooks' not in settings:
    settings['hooks'] = {}

hooks = settings['hooks']

# SessionStart
ss = hooks.get('SessionStart', [])
otel_ss_cmd = "bash /home/agent/.claude/hooks/otel-session-start.sh"
if not any(otel_ss_cmd in str(h) for h in ss):
    ss.append({"type": "command", "command": otel_ss_cmd})
hooks['SessionStart'] = ss

# PostToolUse
ptu = hooks.get('PostToolUse', [])
otel_ptu_cmd = "bash /home/agent/.claude/hooks/otel-post-tool-use.sh"
if not any(otel_ptu_cmd in str(h) for h in ptu):
    ptu.append({"type": "command", "command": otel_ptu_cmd, "matcher": ""})
hooks['PostToolUse'] = ptu

# SessionEnd
se = hooks.get('SessionEnd', [])
otel_se_cmd = "bash /home/agent/.claude/hooks/otel-session-end.sh"
if not any(otel_se_cmd in str(h) for h in se):
    se.append({"type": "command", "command": otel_se_cmd})
hooks['SessionEnd'] = se

with open(path, 'w') as f:
    json.dump(settings, f, indent=2)

print("Hooks registered successfully")
```
